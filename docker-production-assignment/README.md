# Microservices DevOps Platform

## Overview
This project demonstrates a production-grade microservices system using:

- Docker
- Docker Swarm
- Kubernetes
- Istio

## Services
- Account Service (Node.js)
- Transaction Service (Spring Boot)
- Notification Service (Flask)

## Phases
1. Docker Setup
2. Swarm Deployment
3. Kubernetes Deployment
4. Istio Traffic Routing
